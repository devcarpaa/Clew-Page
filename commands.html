DISCORD

Bot Commands

/ban Ban user from the server.
/unban Unban a user from the server by Discord ID.
/kick Remove user from server.
/mute Temporarily mute user from server.
/unmute Remove temporary mute from user.
/temprole Give temporary role to user
/channel Lock/unlock writing to a channel.
/cleardm Delete messages that the bot sent to a user in the DM.
/clear Delete a specific number of messages from a channel.
/key Deliver the cheat key with its information to the user. Send proof automatically.
/account Deliver account information to the user. Send proof automatically.
/rockstar Automatically deliver Rockstar account, with stock control and automatic proof delivery.
/discord Automatically deliver Discord account, with stock control and automatic proof delivery.
/steam Automatically deliver Steam account, with stock control and automatic proof delivery.
.ticketsend Send ticket panel.
.rename Change ticket name.
.add Add user to ticket.
.remove Remove user from ticket.
.addrole Add role to ticket.
.removerol Remove role from ticket.
.verify Send verification system.
.pago Send information about payment methods.
.alias Send payment method information.
.crytpo Send payment method information.
.paypal Send payment method information.
.terms Send the terms and conditions of the server.
.purge Same method as /clear but with prefix.
.partner Send the partner role system.
/conversion Convert foreign currencies to ARS.
/avatar View a user profile picture.
/profile View a user profile.
/giveaway Create a giveaway.
